# Task Manager Mobile Application

A simple Task Management Android application with backend integration.

## Features
- User Login using JWT authentication
- Secure token-based API access
- View tasks from backend
- Clean and minimal UI

## Tech Stack
- Android (Java)
- Retrofit
- Node.js + Express
- MongoDB Atlas
- JWT Authentication

## Setup Instructions

### Backend
1. Clone backend repository
2. Run `npm install`
3. Create `.env` file
4. Start server using `node server.js`

### Android App
1. Open project in Android Studio
2. Run on emulator or device
3. Login using valid credentials

## API Endpoints
- POST `/api/auth/login`
- GET `/api/tasks`
